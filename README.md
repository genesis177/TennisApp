## Описание проекта

Это fullstack приложение для генерации расписания парных теннисных матчей.

Пользователь задаёт параметры турнира (игроки, группы, количество кортов, интервалы и дату старта), после чего система автоматически формирует расписание игр без повторов игроков и с учётом ограничений.

## Как работает система

### 1. Пользователь вводит данные:
- Игроки группы A и B
- Количество кортов
- Количество игр
- Дату старта
- Интервал (неделя / месяц)

### 2. Frontend отправляет запрос:
GET /games

### 3. Backend:
- формирует игроков
- создаёт возможные команды (A × B)
- генерирует матчи
- строит расписание без повторов
- распределяет игры по датам и кортам

### 4. Ответ:
Возвращается JSON с расписанием матчей

---

## Основная идея

Система автоматически решает задачу построения оптимального расписания парных игр, учитывая:
- уникальность игроков в матчах
- баланс между командами
- количество доступных кортов
- временные интервалы
